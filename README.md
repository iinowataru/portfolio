# evaluation foods

## サイト概要
食品レビューサイト

### サイトテーマ
消費者に商品のレビューを行い楽しく買い物ができるようなサイト

### テーマを選んだ理由
前職の経験を関連付けられると考えた為（小売業）

### ターゲットユーザ
小売店で買い物をする主婦、学生層

### 主な利用シーン
新商品のレビュー、既存品の評価等
（商品を新規投稿して消費者のコメントを表示、いいね機能を実装して評価を見える化等）
